# PID 控制器模拟器 (SDL2 实现) 🎮

一个直观演示 PID 控制原理的交互式物理模拟程序，使用现代 C++ 和 SDL2 实现。

## ✨ 核心功能

- **可视化 PID 控制**：实时观察小球追踪目标位置的过程
- **交互式调节**：键盘实时调整 PID 参数
- **物理模拟**：
  - 重力加速度：98 px/s²
  - 弹性碰撞：反弹系数 0.3
  - 固定时间步长：60FPS
- **即时反馈**：屏幕实时显示当前 PID 参数

## 🕹️ 操作指南

| 按键      | 功能描述               |
| --------- | ---------------------- |
| 鼠标点击  | 设置目标高度           |
| ↑/↓       | 调节比例系数 (Kp ±5)   |
| ←/→       | 调节积分系数 (Ki ±0.1) |
| PgUp/PgDn | 调节微分系数 (Kd ±5)   |
| R         | 重置 PID 控制器        |

## 🛠️ 编译运行
